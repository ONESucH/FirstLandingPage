Android Wear 2.0 будет поддерживать автоматический запуск приложений

На конференции для разработчиков I/O 2016 была представлена вторая версий операционной системы Android Wear, которая, по заверениям Google, является крупнейшим обновлением платформы с моменты релиза. Из ключевых изменений версии 2.0 стоит выделить обновлённый пользовательский интерфейс, возможность запуска приложений непосредственно на часах, кастомизацию циферблатов с помощью виджетов и новые методы ввода текста. Вскоре Google рассказала разработчикам и о других интересных нововведениях. Одним из них стала возможность автоматического запуска приложений в зависимости от действий владельца аксессуара.
Например, если пользователь начинает бежать, то на часах автоматически запускается приложение RunKeeper, которое будет отслеживать преодолённое расстояние. Другие программы также будут знать, что владелец в данный момент занимается спортом, и своевременно начнут работу. Таким образом, пользователю больше не придётся перед тренировкой запускать все необходимые приложения вручную.
Стоит отметить, что разработчики должны будут самостоятельно добавить поддержку этой функции в свои программы.

Microsoft добавит возможность панорамной съёмки в приложение камеры на Windows 10 Mobile

Несмотря на тот факт, что компания Microsoft не считает смартфоны своим приоритетом, работа над мобильной системой Windows 10 Mobile идёт непрерывно. Сегодня стало известно о новой функции, которая должна появиться в одном из грядущих обновлений платформы. Пока это неофициальные данные, открывшиеся в результате утечки. Сама функция позволит пользователям снимать панорамные снимки. Источники утверждают, что в будущем функция будет внедрена в стандартное приложение камеры.

Запустить панорамную съёмку можно будет с помощью специальной кнопки непосредственно из приложения камеры. После активации режима пользователь должен двигать смартфон слева направо. Программа автоматически соберёт отснятый материал в один панорамный снимок.
Источники отмечают, что функция съёмки панорам в стандартном приложении камеры будет доступна не всем. Первыми опробовать её смогут владельцы смартфонов из линейки Lumia.
